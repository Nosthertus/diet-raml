Diet Raml Generator

diet-raml-generator is a code generator for resources found in a raml file. Raml version used is 0.8

Version

0.6.4

Installation

$ npm install diet-raml-generator -g

Usage

$ diet-raml-generator -t test/api.raml -d test/routes

Documentation

-d The directory where to save the generated routes.

-t The target raml file to read.
